🩸 Blood Compatibility Guide

O- (Universal Donor)

Can donate to: All blood types

AB+ (Universal Receiver)

Can donate to: AB+ only

A, B, O Positive

Can donate to: Same type + AB+

A, B, AB Negative

Can donate to: Same type ± variants

Can receive from: Negative types only

⚠ Important: Always verify blood compatibility with medical professionals before donation. This guide is for reference only.
